# EyePagE – GOD-LEVEL React Experience 🚀

**Live Site:** [https://book-collection-simple-website.vercel.app/](https://book-collection-simple-website.vercel.app/)

Welcome to the newly overhauled **EyePagE v3.0**. What started as a beautiful static project has now been transformed into a "Product Experience" inspired by the likes of Duolingo, Apple, and Notion.

---

## 🌟 Key Features
- **3D Hero Scene:** Interactive 3D book experience powered by Three.js (React Three Fiber).
- **Paige Mascot:** An expressive companion that tracks your cursor and reacts to your interactions.
- **Micro-Interactions:** Every button, card, and input is alive with animations (Framer Motion).
- **Gamification:** Streak counter and XP system integrated into the global state (Zustand).
- **Day/Night Mode:** A cinema-grade theme switcher that changes colors, stars, and mascot behaviors.
- **Glassmorphism UI:** Premium design with backdrop blurs, rich shadows, and vibrant brand colors.

---

## 🛠️ Tech Stack
- **Framework:** React 18 + Vite
- **Styling:** Tailwind CSS + PostCSS
- **Animations:** Framer Motion + GSAP
- **3D Graphics:** Three.js + React Three Fiber + @react-three/drei
- **State Management:** Zustand (with Persistence)
- **Routing:** React Router v6
- **Icons:** Lucide React

---

## 📂 Project Structure
- **/src/components/character:** Contains `Paige.tsx` (the mascot).
- **/src/components/three:** Contains the 3D book logic.
- **/src/pages:** All site pages (Home, Library, About, Support, Login, Register).
- **/src/store:** Global Zustand state logic.

---

## 🚀 How to Run
To see the magic in action:

1.  Open your terminal in the project directory.
2.  Install dependencies (already done for you): `npm install`
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the link provided (usually `http://localhost:5173`) in your browser.
